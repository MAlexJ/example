 Работа со списком системных свойств

В примере показан способ изменения системных свойств класса Properties
в автономном приложении Java.
С использованием этого способа вы можете расширить список системных свойств,
добавив в него, например, рабочие параметры вашего приложения или любую
произвольную текстовую информацию.
